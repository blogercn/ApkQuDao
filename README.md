# ApkQuDao
使用方法：dzm_qudao.txt配置
配置文件配置：
</br>
app_path=app-release_aligned_signed.apk;
</br>
keystore=ej_v3.jks;
</br>
keyAlias=ej_v3;
</br>
contents=优亿,魅族;
</br>
apkname=youyi,meizu;
</br>
keyPassword=123456;
</br>
storePassword=123456;
</br>

可生成jar包
</br>
通过命令行：
</br>
java -jar ApkQuDao.jar
</br>
ApkQuDao.jar可通过编译项目生成jar文件
